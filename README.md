# 🕐 Digital Clock

> *"Time, rendered in light. The eternal count, made visible."*

A feature-rich digital clock application with world clock support, military time, an alarm system, and a countdown timer. Built with vanilla HTML, CSS, and JavaScript.

**[🔮 View Live](https://stackedalchemist.github.io/Portfolio/digital-clock/)**

---

## ✦ Features

- **Live Digital Clock** — Displays current local time with seconds
- **12h / 24h Toggle** — Switch between standard and military time formats
- **World Clocks** — Display multiple time zones simultaneously
- **Alarm System** — Set alarms with custom labels; triggers audio alert at set time
- **Countdown Timer** — Set a custom duration and count down with visual feedback
- **Responsive Design** — Works cleanly on desktop and mobile

---

## 🛠 Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Structure |
| CSS3 | Animations, responsive layout, dark theme |
| Vanilla JavaScript | Time logic, alarm system, timer |
| Web Audio API | Alarm sound generation |
